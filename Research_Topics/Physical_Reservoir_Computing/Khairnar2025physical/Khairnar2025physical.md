# Khairnar2025physical — Physical Reservoir Computing with Compliant Fiber Networks

## 1. Core contribution

This paper studies compliant fiber networks as mechanical reservoirs inspired by web-like architectures. It argues that distributed deformation, tension, and wave propagation in such networks can provide useful memory and nonlinear processing. The contribution is a combined modeling and physical demonstration of fiber-network PRC. It expands mechanical reservoirs beyond compact bodies and rigidly defined resonators.

## 2. Method / mechanism / evidence

The authors use Cosserat-rod-based simulations to evaluate nonlinear and memory capacities across network choices, then build physical prototypes to test the basic computational behavior experimentally. The work examines how topology, material response, pretension, and network scale affect information processing. Reported experiments show nonlinear transformation and recall-like behavior from the fabricated structure. The simulation and hardware results point in the same direction while also revealing differences that need further explanation.

## 3. Key takeaway and limitation

The main takeaway is that sparse compliant networks can act as physically interpretable reservoirs whose computation is distributed through deformation. This makes them interesting for sensing-rich mechanical systems. The main limitation is that scaling trends and quantitative memory behavior do not align perfectly across simulation and experiment, so the design rules are not yet settled. The paper establishes a promising substrate rather than a fully optimized architecture.

## 4. Open question

Which mechanical parameters most reliably control the memory-versus-nonlinearity profile of fiber-network reservoirs in real fabricated systems?
